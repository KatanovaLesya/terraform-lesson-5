# Terraform Lesson 5

Проєкт створює інфраструктуру AWS:

- **S3 + DynamoDB** для Terraform backend
- **VPC** з публічними та приватними підмережами
- **ECR** для зберігання Docker-образів

## Команди для запуску

```` ```bash ````

terraform init
terraform plan
terraform apply
terraform destroy
